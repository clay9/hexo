* 功能
  - 错误单词高亮显示
  - 自动correct words
    flyspell-auto-correct-words
    flyspell-correct-words
  - 手动检查错误统计
    flyspell-region 
    flyspell-buffer

* 安装
  1. flyspell-mode是font-ends, emacs自带
  2. 需要安装back-ends, 推荐aspell
     sudo apt-get install aspell
